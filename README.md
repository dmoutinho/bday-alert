# Bday Alert

### Prerequisites

- Maven in 5 Minutes
https://maven.apache.org/guides/getting-started/maven-in-five-minutes.html

### Clone

```sh
git clone https://github.com/dmoutinho/bday-alert.git
```

### Build

```sh
$ cd bday-alert
$ mvn clean install
```

### Run

```sh
$ cd target
$ java -Dserver.port=9080 -jar bday-alert-0.0.1-SNAPSHOT.jar
```
